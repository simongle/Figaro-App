== README

This is a Rails app designed to connect clients and barbers. It will feature payment processing via First Data, and the Full Calender jQuery gem

Things you may want to cover:

* Ruby version 2.2.1

* Rails version 4.2.3

* This uses SQLite3 in development, and PostgreSQL in production

* Extensive tests exist for the user model, to run them use: bundle rake test:model

* Update: 7/20 contains user Model and test suite
